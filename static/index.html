GO JSON Server

A lightweight fake JSON API server implemented in Go(lang). Inspired and converted to Go from the original; json-server by typicode. Created as a small learning project during my time at Kodehode, a web developer course in Norway.

Made with no external dependencies - No need to install anything if you clone/fork the project.

Available Endpoints

  • GET /{collection} - List all entries within a collection
  • GET /{collection}/{id} - Get a single entry within a collection
  • POST /{collection} - Create a new collection
  • PUT /{collection}/{id} - Replace (or create) an entry within a collection
  • PATCH /{collection}/{id} - Update an entry within a collection
  • DELETE /{collection}/{id} - Delete an entry within a collection

Health Check: GET /Health (Only really relevant if hosted somewhere)
